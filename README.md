# Amazon-Clone
This project is a front-end Amazon clone built using HTML and CSS, designed to replicate the look and layout of the Amazon homepage.The project was created to practice core web development skills such as semantic HTML, CSS flexbox/grid, without using JavaScript or backend functionality.
